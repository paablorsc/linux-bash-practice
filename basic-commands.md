# Bash Commands Practice

## Navigation
- `ls`, `cd`, `pwd`
- `tree` to view folder structures

## Permissions
- `chmod 755 file`
- `chown user:group file`
- `ls -l` to check permissions

## Searching
- `find . -name "*.txt"`
- `grep "text" file.txt`

## Pipes and Redirects
- `ls -l | grep ".sh"`
- `echo "Hello" > file.txt`
- `cat file.txt >> log.txt`
